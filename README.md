# KitearepaAgent

Agente de arbitraje automatizado entre moneda local (VES) y USDT en exchange P2P venezolano. Sin intervención humana.

Monitorea, compra y vende USDT automáticamente generando ganancia diaria. Cada ciclo dura mínimo 15 horas. Con Kite Chain: 24/7.
